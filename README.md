# Next.js with TypeScript and TailwindCSS

This is a starter template for [Learn Next.js](https://nextjs.org/learn).

## Get start

```
# Install dependencies
npm install

# Enable husky
npm run prepare

# Start dev server
npm run dev
```

## Feature

- React 18
- TypeScript 4
- TailwindCSS 3
- Jest
- Husky
- Prettier

### devServe.js

Check the [file](./devServe.js) to configure your proxy.
